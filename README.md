# Easy English Angular

Project frontend Angular cho Easy English.

## Yêu cầu

- Node.js
- npm

## Cài đặt

Chạy lệnh sau để cài các package:

```bash
npm install
```

## Chạy project

Chạy server local:

```bash
npm start
```

Sau khi chạy thành công, mở trình duyệt tại:

```text
http://localhost:4200
```

## Build

```bash
npm run build
```

File build sẽ nằm trong thư mục `dist/`.

## Test

```bash
npm test
```
